This project presents a Bike Sales Analysis Dashboard created using Excel to analyze customer purchasing behavior and identify factors influencing bike purchases.

The dashboard helps uncover insights related to:

Customer demographics
Income levels
Commute distance
Age groups
Education and occupation
Regional purchasing trends

The project transforms raw sales data into meaningful business insights through data cleaning, pivot tables, and interactive dashboard visualizations.

## 🎯 Project Objectives

Analyze customer bike purchasing behavior
Identify key factors influencing sales
Compare purchase trends across demographics
Create an interactive business dashboard
Generate insights for better marketing decisions
