## 🎵 Spotify Music Data Analysis
## 📌 Overview

This project analyzes Spotify music data using Python, Pandas, NumPy, Seaborn, and Matplotlib to uncover trends in track popularity, artist performance, album releases, follower counts, and music characteristics.

The analysis focuses on transforming raw Spotify datasets into meaningful insights through data cleaning, exploratory data analysis (EDA), and data visualization.

## 🎯 Project Objectives
Analyze track and artist popularity
Identify top-performing artists and albums
Explore music trends and release patterns
Compare explicit vs non-explicit tracks
Visualize Spotify data using charts and heatmaps
Perform data cleaning and preprocessing
## 📂 Datasets Used

The project uses two Spotify datasets:

spotify_data clean.csv
track_data_final.csv

These datasets include:

Track details
Artist information
Album metadata
Popularity scores
Followers count
Genres
Release dates

## 🧹 Data Cleaning & Preprocessing

Performed several preprocessing steps to improve data quality:

Checked missing values using .isnull()
Filled missing artist and track information
Converted date columns into datetime format
Extracted release year from album release dates
Converted popularity columns into numeric format
Verified dataset structure using .info()

## 🔍 Key Insights
Taylor Swift dominated across popularity, followers, and album creation.
The Weeknd and Ariana Grande ranked among the top-performing artists.
Non-explicit tracks showed higher overall popularity compared to explicit tracks.
Album releases significantly increased during recent years.
Certain albums contained exceptionally large track collections.
Artist followers strongly influenced track popularity trends.
